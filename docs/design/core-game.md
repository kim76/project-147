# Core Game Direction

## Intent

Project 147 is an original mobile tower-defence game with two perspectives:

- defence mode, where players place and upgrade towers;
- alien mode, where players build and upgrade attack waves.

The game can be inspired by classic tower-defence pacing and readability, but it must not copy another game's name, art, maps, characters, UI, specific mechanics or marketing language.

## Defence Mode

The player protects a base from incoming aliens.

Core actions:

- place towers;
- shape paths without fully blocking them;
- upgrade towers;
- trigger abilities;
- survive waves;
- earn rewards.

Tower upgrade examples:

- fire rate;
- damage;
- crit chance;
- range;
- area damage;
- slow strength;
- armour piercing;
- special ability cooldown.

## Alien Mode

The player plans attacks against defensive layouts.

Core actions:

- choose alien squads;
- upgrade alien traits;
- time special abilities;
- counter tower types;
- break through with limited resources.

Alien upgrade examples:

- health;
- speed;
- resistance;
- dodge;
- shield;
- regeneration;
- stealth;
- stun resistance;
- swarm bonuses.

## Special Abilities

Special abilities should create memorable moments without replacing strategy.

Examples:

- tower overcharge;
- temporary slow field;
- orbital strike;
- decoy beacon;
- alien shield pulse;
- tunnel burst;
- tower jammer;
- emergency base repair.

## Humour

Humour should come from reactions, ability names, animations, failure states and upgrade flavour, not from making the UI noisy.

The game should stay readable and tactical during play.
