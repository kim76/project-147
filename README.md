# Project 147

Project 147 is the temporary working name for an original mobile tower-defence game inspired by the broad genre, not a clone of any existing game.

The intended direction is a Unity mobile game for iOS and Android with two playable perspectives:

- defence mode, where the player places and upgrades towers;
- alien mode, where the player builds and upgrades attacking waves.

The project values small tested changes, clear architecture, original art direction and maintainable systems from the start.

## Current Status

Planning and foundation only. No Unity project has been generated yet.
